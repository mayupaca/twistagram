# Twistagram
You need [Twistagram frontend repository](https://github.com/mayupaca/twistagram-frontend/edit/master/README.md) to run this project.

## Backend Tech stack
- Node.js
- Express
- MongoDB
- Postman
- mongoose
- nodemon
- multer(file upload)
